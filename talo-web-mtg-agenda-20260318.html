ログハウス市場自体が縮小傾向にある背景：

  • 建築コスト1.5倍 ― 木材価格・輸送費高騰で、かつて2,000万円台だった物件が3,000万円台に
  • 「憧れで買える商品」→「高級品」へ変質 ― 30代若年層の購買力低下でメイン層が50代以上に偏重
  • 検索ボリュームの縮小 ― 「ログハウス」で検索する母数自体が減少傾向
  • CPC上昇 ― 住宅業界全体でGoogle広告の単価が上昇（AI自動入札の影響も）

月8,000クリック・日300回弱の現実

この表示回数で「確実に取れる見込み客」を増やすには？

月8,000クリック × CVR 0.59% = 月47件のCV（現状）
もしCVR 1.16%（Instagram水準）に改善できれば → 月93件のCVに倍増

つまり、表示回数を増やすよりも、CVRを上げるほうが現実的。
具体的には「誰に見せるか」（ターゲティング精度）と「何を見せるか」（LP・クリエイティブ）の問題。

🔑 3. キーワード戦略 ― ログハウスKWにとどまるか、広げるか

選択肢A：ログハウスKWに留まる（現状維持＋深掘り）

メリット
  • TALO 4位という確立されたポジション
  • 競合が少ない（CPC相対的に低い）
  • 検索者の意図が明確（ログハウスが欲しい人）
  • ブランド指名検索もある程度ある
デメリット
  • 市場自体が縮小中（Vol 7,900で頭打ち）
  • 全トラフィックの60%が「ログハウス」1語に依存
  • 新規顧客の流入天井が見えている
  • 価格上昇でCV率が今後さらに低下する恐れ
深掘り案：ロングテール強化

「ログハウス 価格」(Vol 1,000, 現6位)、「ログハウス 平屋」(Vol 150, 現7位) など、購入意図の強いロングテールKWでのポジション改善に集中。市場は小さいがCVRが高いため、限られた予算での効率は最も良い。

選択肢B：隣接KWへ拡張（木の家・自然素材・北欧住宅 etc.）

議論ポイント：レッドオーシャン問題

「注文住宅」「平屋」は月10万回以上の巨大市場だが、住友林業・一条工務店・積水ハウス等がCPC 500〜1,000円超で入札。TALOの予算では月1,000クリックも確保できない可能性が高い。

一方、「北欧住宅」「サウナ 家」「二拠点生活」はブルーオーシャン。Volは小さいが、競合が少なくCPCも低い。TALOのブランドストーリーとも合致する。

選択肢C：KWではなく「ターゲット人物像」で攻める

発想の転換案

KWを変えるのではなく、「誰に届けるか」を変える。

従来のターゲット：「ログハウスが欲しい人」→ 母数が減少中
新しいターゲット：「ライフスタイルを変えたい人」→ 母数は増加中

具体的には：
・リモートワークで場所に縛られなくなった40〜50代
・サウナに凝り始めた30〜40代
・老後の住まいを考え始めた50〜60代
・セカンドハウス/別荘を検討している富裕層

→ Instagram広告の高CVR（1.16%）はまさにこの層に刺さっている証拠。
検索KWに頼らず、SNS広告でライフスタイル訴求するのが現実的な第3の道。

SEO中断の影響は？

現状、「ログハウス」4位、「ログハウス 内装」「ログハウス キッチン」等で1位を維持。
短期的にはポジション維持されているが、中断が続くとDR低下→順位低下のリスクあり。
特にBESSはDR 53でコンテンツ投下を継続しており、差が開く一方。

ただし、SEOは即効性がないため、限られた予算では広告最適化が先。
SEOは「やめる」のではなく「最小コストで維持する」が現実的。

GEO（生成エンジン最適化） ― AI時代のSEO

2026年、SEOは大きな転換期を迎えています。Google AI Overview、ChatGPT Search、Perplexity等の生成AIが検索結果を変え始めています。

従来のSEOGEO（生成エンジン最適化）
検索結果1ページ目に表示AIの回答に自社が「引用」される
キーワード密度・被リンク重視構造化データ・専門性・信頼性重視
クリックしてサイトに来てもらうAIに選ばれ「指名検索」を誘発する
競合は同業他社競合はAIが参照する全ての情報源
TALOが今すぐ着手すべきGEO対策

1. 構造化データ（Schema.org）の実装
→ 商品情報、FAQ、施工事例をAIが読み取れる形式に

2. E-E-A-T（経験・専門性・権威性・信頼性）の強化
→ 「フィンランド直輸入40年の実績」「施工実績1,200棟」等の具体的数値をサイトに明記

3. FAQ/ナレッジベースの充実
→ 「ログハウスとは」「メンテナンス方法」「価格の目安」等をQ&A形式で整備
→ AIが回答生成時に引用しやすい形式にする

4.「ログハウスといえばTALO」の指名検索を増やす
→ GEO時代は「AIに選ばれる → 指名検索で来訪」が主流導線に

コスト：低（サイト内コンテンツ整備が中心）

🏗️ 6. サイト改修・LP・差別化 ― 優先順位の決定

3つの施策の優先順位

施策効果コスト即効性優先度
① LP制作（広告専用） 高中（50〜100万円）即効 最優先
② 差別化コンテンツ追加 中〜高低中期 優先度2
③ サイト全面改修 高高（300万〜）半年〜 後回し

① LP制作が最優先である理由

現状の問題

広告 → トップページ に流している場合、ユーザーの検索意図と着地ページの内容がミスマッチしている可能性が高い。

例：「ログハウス 価格」で検索 → トップページに来る → 価格情報がすぐ見つからない → 離脱

広告KW別の専用LPを作れば：
・「ログハウス 価格」→ 価格一覧LP → CVR 2〜3倍に
・「ログハウス 平屋」→ 平屋施工事例LP → CVR 2〜3倍に
・Instagram広告 → ライフスタイル訴求LP → CVR維持〜改善

LP1本で月間CV数が大幅に変わる。サイト全面改修の1/5のコストで、広告効果は直ちに改善。

② 他社との差別化ポイント

TALOが打ち出すべき差別化要素（サイト内で不足しているもの）：

差別化要素TALOの強み現サイトでの訴求

40年以上の実績施工実績の信頼性不明確
カスタマイズ自由度完全注文設計が可能伝わりにくい
サウナとの親和性フィンランド式サウナ×ログハウス未訴求
リセールバリューログハウスの経年美化未訴求
全国対応代理店ネットワーク普通

→ 特に「サウナ×ログハウス」はHonkaが先行しているが、TALOは未着手。大きなチャンス。
